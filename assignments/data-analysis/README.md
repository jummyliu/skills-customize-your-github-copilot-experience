# 📘 Assignment: Data Analysis

> 🟡 **难度 / Difficulty：中级 (Intermediate)**

## 🎯 Objective

Students will learn the basics of data analysis using Python. They will load, explore, and analyze a dataset to extract meaningful insights.

## 🎯 目标

学生将学习使用 Python 进行数据分析的基础知识。他们将加载、探索并分析数据集，从中提取有意义的洞察。

## 📝 Tasks / 任务

### 🛠️ Data Loading and Exploration / 数据加载与探索

#### Description
Load a provided CSV dataset and perform basic exploration to understand its structure and contents.

#### 描述
加载提供的 CSV 数据集，并进行基本探索以了解其结构和内容。

#### Requirements
Completed program should:

- Load a CSV file using Python (e.g., with pandas)
- Display the first 5 rows of the dataset
- Show summary statistics (mean, median, etc.) for numeric columns

#### 要求
完成的程序应满足：

- 使用 Python 加载 CSV 文件（例如使用 pandas）
- 显示数据集的前 5 行
- 显示数值列的汇总统计信息（均值、中位数等）


### 🛠️ Data Visualization and Insights / 数据可视化与洞察

#### Description
Create visualizations to help understand the data and summarize key findings.

#### 描述
创建可视化图表以帮助理解数据，并总结关键发现。

#### Requirements
Completed program should:

- Generate at least two different types of plots (e.g., histogram, scatter plot)
- Identify and describe at least two insights or trends from the data
- Save the plots as image files

#### 要求
完成的程序应满足：

- 生成至少两种不同类型的图表（例如直方图、散点图）
- 从数据中识别并描述至少两个洞察或趋势
- 将图表保存为图像文件

## 🚀 Advanced Challenges / 进阶挑战

> 完成以上任务后，尝试以下进阶挑战来加深理解！

1. **Data Cleaning / 数据清洗**：Identify and handle missing values and duplicate rows in the dataset. Describe your strategy (drop, fill, etc.) and explain why. / 识别并处理数据集中的缺失值和重复行，说明你的处理策略（删除、填充等）并解释原因。

2. **Correlation Analysis / 相关性分析**：Compute and visualize the correlation matrix for numeric columns. Identify the strongest positive and negative correlations. / 计算并可视化数值列的相关矩阵，找出最强的正相关和负相关关系。

3. **Groupby Analysis / 分组分析**：Group the data by a categorical column and compute aggregate statistics (mean, sum, count) per group. Present the results in a clear table or bar chart. / 按某个分类列对数据进行分组，计算每组的聚合统计（均值、求和、计数），并以清晰的表格或条形图展示结果。
